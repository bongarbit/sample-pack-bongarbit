# Sample Pack Bongarbit

![](<Todos no Carranca.jpg>)

Gravado no **Estúdio Carranca**, Recife, Pernambuco - Brasil em **29 de setembro de 2023**.

Este Sample Pack é uma primeira amostra simplificada de uma sessão de gravação complexa, com mais resolução, mais instrumentos e mais variações de timbres e microfones. Foi idealizado como projeto de Design de Som para os instrumentos digitais construídos pelo coletivo **Bongarbit**: https://www.instagram.com/bongarbit

Desenvolvido como parte do projeto **Instrumentos Digitais para o Carnaval de Rua Pernambucano** (Proj. nº 722/2020), contemplado pelo **4º Edital do Programa de Fomento à Produção em Música de Pernambuco – Funcultura 2019/2020**.
